NPPS4 Android
=====

This repository contains build recipe of [NPPS4](https://github.com/DarkEnergyProcessor/NPPS4)
for Android platform as standalone Android app with help of Chaquopy.

This effectively replaces "Termux" way of running NPPS4.

Downloads
-----

Go head to the releases page. Do note that currently releases are in form of v0.0.yyyymmdd due to
NPPS4 nature being a rolling release at the moment.

The app requires Android 7.0 or later running ARM64 or x86-64 CPU.
